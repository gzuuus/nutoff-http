About This Service

This server provides LNURL-PAY functionality for email-like addresses, following the LUD-16 specification. It acts as a proxy to resolve LNURL addresses using MCP (Model Context Protocol) servers.

Features

  • LNURL-PAY endpoint for Lightning payments
  • MCP server integration for address resolution
  • Health check endpoint for monitoring
  • RESTful API for Lightning invoice generation

Available Endpoints

GET /.well-known/lnurlp/:username

LNURL-P endpoint for payment requests

GET /lnurlp/callback/:username?amount=:amount

Callback endpoint for invoice generation

GET /health

Health check endpoint
